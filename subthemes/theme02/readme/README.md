# Theme 02 - Masseria Le Cerase

Theme 02 is generated from the clean Masseria Le Cerase HTML package and adapted to the SyncBooking WordPress bridge.

## Structure

```text
theme02/
├── data_general.php
├── index.php
├── villa.php
├── house.php
├── whole-masseria.php
├── price-and-condition.php
├── spa-wellness.php
├── experiences.php
├── weddings.php
├── offers.php
├── surroundings.php
├── contacts.php
├── header/
│   └── header.php
├── footer/
│   └── footer.php
├── functions/
│   ├── helpers.php
│   ├── house-detail.php
│   └── simple-page.php
└── assets/  (not bundled; downloaded online from assets.zip)
```

## Rules

- Keep shared header markup in `header/header.php`.
- Keep shared footer markup in `footer/footer.php`.
- Keep helpers and reusable partials in `functions/`.
- Do not bundle demo media, CSS, JavaScript, video or brochure assets.
- Assets are downloaded online from `https://syncbooking.com/clone-theme/theme-02/assets.zip`.
