# SyncBooking Villa Resort

Tema WordPress hospitality per ville, resort, case vacanza e strutture con motore di prenotazione esterno.

## Installazione

1. Scarica il repository come ZIP da GitHub.
2. Caricalo in WordPress da `Aspetto > Temi > Aggiungi nuovo > Carica tema`.
3. Attiva `SyncBooking Villa Resort`.
4. Configura logo, menu e dati booking da `Aspetto > Personalizza > Resort e Booking`.

## Contenuti inclusi

- Home page completa in `front-page.php`.
- Form disponibilità riusabile con shortcode `[syncbooking_form]`.
- Menu principale e menu footer.
- Campi Customizer per URL booking, telefono, email, indirizzo e immagine hero.
- Layout responsive per resort, case, servizi, SPA, esperienze e offerte.

## Nota sugli asset

Il tema usa immagini remote dimostrative e testi originali. Sostituiscili dal Customizer o nei template con foto e contenuti ufficiali della struttura.
